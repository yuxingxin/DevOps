#### 项目设置

对于项目设置，各个用户受权限不同，所显示的功能也不一样，总体分六大部分：

- 常规：主要是针对项目本身的资料进行设置。
- 成员：主要是对项目成员进行角色分配管理。
- 集成：可以集成三方应用，利用Webhooks添加触发器。
- 版本库：主要设置项目保护分支和保护标签，以及部署密钥。
- 流水线：主要用于GitLab CI，执行我们设置的任务。
